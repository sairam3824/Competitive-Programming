2. Setup your shell environment:

   MAC / LINUX (Zsh/Bash):
   Add this to your ~/.zshrc or ~/.bashrc:
   source /path/to/Competitive-Programming/.zshrc_competitive_programming

   Then run:
   source ~/.zshrc
   OR
   If in folder 
   source .zshrc_competitive_programming

   WINDOWS:
   Option 1 (Recommended): Use WSL or Git Bash and follow Mac/Linux instructions above.
   Option 2 (PowerShell): You'll need to manually run the listener and compile commands, or set up similar aliases in your PowerShell profile.

3. Start listener:
   cflisten  (if you set up the shell alias)
   OR
   ./start-listener
